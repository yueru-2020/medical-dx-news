# 医療DXニュースキュレーション・自動化システム設計

## 1. 概要
毎朝指定した時間に日経新聞から最新の医療DX記事を5件取得し、要約、Webサイト更新、およびLINEへの通知を自動で行うシステムです。

## 2. システム構成
- **収集・処理**: Pythonスクリプト (`update_news.py`)
    - `Playwright` または `BeautifulSoup`: 日経新聞からの記事取得
    - `OpenAI API (GPT-4o)`: 編集者目線での3行要約
- **表示**: HTML/CSS (`index.html`)
    - Pythonスクリプトにより自動生成（テンプレートエンジン `Jinja2` を使用）
- **通知**: LINE Notify API または LINE Messaging API
    - 更新完了時に要約とリンクを送信
- **自動化**: Windows タスクスケジューラ
    - 毎日決まった時間（例: 朝7:00）に実行

## 3. 実装のステップ
1. **Python環境の構築**: 必要なライブラリのインストール。
2. **スクレイピングロジックの実装**: 最新記事5件のURLとタイトルを抽出。
3. **要約ロジックの実装**: AIを使用して指定のフォーマットで要約。
4. **HTML生成ロジックの実装**: 取得したデータをHTMLに埋め込み。
5. **LINE通知の実装**: API経由でメッセージ送信。
6. **スケジュール設定**: Windowsでの定時実行設定。

## 4. 必要な準備
- OpenAI API Key (要約用)
- LINE Notify Token (通知用)
